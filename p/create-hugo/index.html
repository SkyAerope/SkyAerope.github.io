Hugo搭建参考资料

记下参考了哪些资料来搭建，以防自己忘记

Apr 04, 2021

参考资料

主题： https://github.com/CaiJimmy/hugo-theme-stack
主题官方文档： https://docs.stack.jimmycai.com/v/zh-cn/

安装主题后，启动一个临时服务器（不要正式使用）
$ hugo server –theme=hugo-theme-stack

解决无法启动问题（删除根目录config.toml即可）： https://github.com/CaiJimmy/hugo-theme-stack/issues/105

hugo server启动模式（改端口什么的）： https://www.jianshu.com/p/2c6aa941f657

辅助资料

官方中文文档： https://www.gohugo.org/

参考末尾"在站点根目录执行 $ hugo 命令生成最终页面，所有静态页面都会生成到 public 目录"

完全使用过程： https://zhuanlan.zhihu.com/p/126298572

重点参考"2.Hugo 安装"“7.编写博客文章”

备用资料

标题：“通过 Hugo+Nginx 部署静态网页”： https://blog.csdn.net/Homography/article/details/106168190
